# Dog gallery

Dog Gallery is an Android SDK client to https://dog.ceo/dog-api service. The app features are:

  - Search dog pictures randomly
  - Search dog pictures by breed
  - Show picture details
  - Dark mode (TO-DO)
  - Share dog picture (TO-DO)
